## What RepoTalk Does

**RepoTalk is a chat app that helps you talk to computer code.**

Imagine you want to understand someone's project on GitHub (a website where programmers share their work), but you don't know how to read code. RepoTalk lets you ask questions in plain English, like "What does this project do?" or "How does the login work?" and it explains everything in simple terms.

**Who Uses It:**
- Business owners who want to understand their team's code
- Managers reviewing projects
- Students learning about programming
- Anyone curious about how a piece of software works

Think of it like having a friendly translator who speaks both "computer language" and "human language." You paste a link to any public GitHub project, then chat with it as if you're texting a knowledgeable friend who can explain the code to you.
